About This Server

Project Overview

This is a concurrent HTTP web server developed as part of a Systems Operations course.

Architecture

  • Master Process: Manages initialization and IPC
  • Worker Processes: Handle incoming connections
  • Thread Pools: Process HTTP requests concurrently
  • IPC: Shared memory and semaphores for synchronization

Features

  • HTTP/1.1 GET request support
  • Multi-process, multi-threaded architecture
  • Connection queue management
  • Request caching
  • Comprehensive logging
  • Performance statistics
